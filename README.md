# Kalpit Pokra - 20161134

3-D Game (Legend of Zelda) made using openGL3 (C++).

### Run from terminal

- cd <dir>
- mkdir build && cd build
- cmake ..
- make
- ./graphics_asgn1

### Controls

- **Up Key** - Move forward
- **Down Key** - Move backward
- **SPACE** - Jump
- **F** - Fire
- **C** - Change Camera
